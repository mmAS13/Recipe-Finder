#  Recipe Finder App

A modern and responsive Recipe Finder Web App built using HTML, CSS, and Vanilla JavaScript that allows users to search for delicious recipes from around the world.

This application uses the TheMealDB API to fetch real recipe data including food images, ingredients, and cooking instructions, making it a simple yet powerful cooking companion.

The project demonstrates how to work with APIs, dynamic UI rendering, and responsive layouts in a real-world frontend application.


# Features

1. Search Recipes
Users can search for any food dish and instantly get matching recipes.

2. Recipe Cards with Images
Each recipe appears as a beautiful card displaying:
Recipe image
Recipe name
Category or cuisine
View recipe option

3. Recipe Details
Users can open a detailed view of the recipe showing:
Large food image
Ingredients list
Cooking instructions

4. Responsive Design
The layout adapts perfectly to mobile, tablet, and desktop devices.

5. Real API Data
Recipes are fetched in real-time using the TheMealDB API.
